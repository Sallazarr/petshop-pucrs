# Projeto PetShop - Fase 1

**Disciplina:** Fundamentos de Sistemas Web - PUCRS



## Funcionalidades

Este sistema web apresenta a interface inicial de um PetShop, contendo:

- **Catálogo de Produtos:** Dividido em Acessórios, Rações e Higiene.

- **Detalhamento de Serviços:** Listagem de Banho e Tosa com informações sobre Tele-busca.

- **Navegação Semântica:** Uso de âncoras para navegação rápida entre as seções.



## Tecnologias Utilizadas

- HTML5 (Estrutura e Marcação).

- Hospedagem: GitHub Pages.

# Projeto PetShop Salazar - Fase 2 (Final)

**Disciplina:** Fundamentos de Sistemas Web - PUCRS 

## Funcionalidades

Este sistema web representa a evolução final da plataforma, integrando interatividade e design responsivo:

- **Catálogo Completo de Produtos:** Apresentação de 6 itens divididos em Acessórios, Rações e Higiene, com imagens, descrições e valores.
- **Agendamento Inteligente:** Formulário com validação de data para impedir seleções retroativas e restrição de horários comerciais.
- **Resumo em Tempo Real:** Painel lateral que atualiza instantaneamente o serviço e o método de busca selecionados através de manipulação do DOM via JavaScript.
- **Status de Atendimento Automático:** Sistema que identifica se a loja está aberta ou fechada com base no horário real do sistema.
- **Acessibilidade Proativa:** Inclusão de modo de alto contraste para usuários com baixa visão e links de salto (**skip-links**) para navegação por teclado.
- **Interface Responsiva:** Layout que se adapta automaticamente a celulares, tablets e desktops utilizando o **grid system** do Bootstrap.

## Tecnologias Utilizadas

- **HTML5:** Marcação semântica para melhor SEO e estrutura de acessibilidade.
- **CSS3 Personalizado:** Uso de variáveis nativas e estilização avançada para componentes como **cards** e **hero section**.
- **Bootstrap 5.3:** Framework utilizado para agilizar a responsividade e componentes visuais como Carrossel e Navbars.
- **JavaScript (ES6 Modules):** Organização do código em módulos independentes (`relogio.js`, `acessibilidade.js`, `formulario.js`) para garantir a manutenibilidade.
- **Git & GitHub:** Utilização para gestão de configuração de software (GCS) e controle de versão.
- **Hospedagem:** Publicação contínua diretamente via GitHub Pages.
